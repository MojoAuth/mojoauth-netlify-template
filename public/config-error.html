Configuration Error

Missing required environment variables for MojoAuth authentication.

Environment Variable Status:

MOJOAUTH_CLIENT_ID: Missing
MOJOAUTH_CLIENT_SECRET: Missing
MOJOAUTH_REDIRECT_URI: Missing
MOJOAUTH_ISSUER: Using default (https://api.mojoauth.com)

How to Fix This:

  1. Create a MojoAuth account at mojoauth.com if you haven't already.
  2. Create an application in the MojoAuth dashboard and enable OIDC.
  3. In your Netlify dashboard, go to Site settings > Environment variables.
  4. Add the following environment variables with values from your MojoAuth application:
    • MOJOAUTH_CLIENT_ID
    • MOJOAUTH_CLIENT_SECRET
    • MOJOAUTH_REDIRECT_URI (typically https://your-site.netlify.app/.netlify/functions/auth-callback)
  5. Redeploy your site or restart your Netlify dev server.
